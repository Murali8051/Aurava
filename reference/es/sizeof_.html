Nombre

sizeof()
Ejemplos 
char arr[10]; // Arreglo de 10 caracteres
int val = 1000;

void setup() 
{
  Serial.begin(9600);  
  Serial.print("Array size in bytes is: ");
  Serial.println(sizeof(arr));
  Serial.print("val size in bytes is: ");
  Serial.println(sizeof(val));
  Serial.print("double datatype size in bytes is: ");
  Serial.println(sizeof(double));
}

void loop() 
{
  
}
Descripción El operador sizeof se usa para determinar la cantidad de espacio en bytes que un dato-elemento/tipo de dato ocupa ne memoria. Para usar sizeof, la palabra reservada "sizeof" esta acompañada del nombre de un tipo de dato, variable, o expresion. Si se usa un tipo de dato, este necesita siempre estar encerrado en paréntesis, mientras que los nombres de variables pueden ser especificados con o sin paréntesis.
Sintaxis 
sizeof var
sizeof(var)
sizeof(datatype)
Parámetros
var nombre de una variable
datatype cualquier tipo de dato válido: int, double, float, char, double
Retorna int: el tama√±o en bytes de la variable o tipo de datos
