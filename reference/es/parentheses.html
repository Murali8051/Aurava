Nombre

() (parentheses)
Ejemplos 
int a;
a = (4 + 3) * 2;          // Agrupando expresiones
if (a > 10) {          // Conteniendo expresiones
  digitalWrite(8, HIGH);  // Conteniendo una lista de par&aacute;metros
}
Descripción Agrupando y conteniendo expresiones y parámetros. Los paréntesis tienen multiples funciones relacionadas a las funciones y las estructuras. Son usadas para contener una lista de parámetros tanto en funciones como en estructuras de control y son usadas para agrupar expresiones de control con el fin de controlar el orden de ejcución. Algunas funciones no tienen ningún parámetro y en este caso, el espacio entre los paréntesis queda en blanco.
Sintaxis 
función()
función(p1, ..., pN)
estructura(expresión)
Parámetros
función Cualquier función
p1...pN Lista de parámetros específica de la función
estructura Estructuras de control como if(), for(), while()
expresiones Cualquier expresión o grupo de expresiones válidas
