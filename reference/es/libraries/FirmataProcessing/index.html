Wiring/Processing usando el protocolo Firmata

Esta librería permite controlar la tarjeta Wiring desde Processing sin escribir código en el Wiring IDE. En cambio, deberá cargar el archivo StandardFirmata ubicado en File->Examples->Firmata->Example.

La librería Processing/Firmata puede ser descargada aquí

Para instalar la librería ubique la carpeta Wiring en la carpeta libraries dentro de Processing.

Wiring()
Crea el objeto Wiring.

list()
Imprime la lista de dispositivos disponibles.

pinMode()
Asigna un pin digital en modo INPUT o OUTPUT.

digitalRead()
Retorna el valor de un pin digital, asignado en modo INPUT.

digitalWrite()
Escribe HIGH o LOW en un pin digital.

servoWrite()
Permite la manipulación de un servo motor.

analogRead()
Retorna el valor de un pin analogo.

analogWrite()
Escribe un valor analogo (PWM) en un pin digital que lo soporte.
