Referencia para la versión de Wiring 1.0 Build 0100+. Si tiene una versión previa, use la referencia incluida con su software. Si encuentra errores o tiene comentarios, no dude en contactarnos.

Nombre

Wiring
Ejemplos 
//Import the Serial i/o and the Wiring Libraries
import processing.serial.*;
import Wiring.*;
//Create a Wiring object
Wiring wiring;
void setup() {
  //print the serial devices list
  println(Wiring.list());
  //inicialized the wiring object
  wiring = new Wiring ( this, Wiring.list()[1], 57600); 
  //sets the pinMode
  wiring.pinMode(48, Wiring.OUTPUT);
}
void draw() {
  //.........
}
Descripción El método Wiring() crea el objeto Wiring, el cual inicia la comunicación. Nota: El protocolo firmata posee una velocidad mínima de 5700 bauds.
Sintaxis 
Wiring(parent, nombre, velocidad)
Métodos
list()
pinMode()
digitalRead()
digitalWrite()
analogRead()
analogWrite()
servoWrite()
Parámetros
parent El parent debería ser "this".
nombre El nombre del dispositivo.
velocidad La velocidad de la conección. Para el protocolo Firmata la velocidad mínima es 57600.
