LiquidCrystal

La librería LiquidCrystal de Wiring permite facilmente escribir datos en una liquid crystal display (LCD) genérica usando el controlador H44780. Puede ser usado como output de tu programa de Wiring.

El display LCD tiene 14 o en algunos casos 15 pines, de los cuales 8 son usados para transmitir datos, 3 para controlar operaciones, 2 para corriente, 1 para controlar 1 el contraste del display y uno opcional para la iluminación.

 

LiquidCrystal
Clase para manipular displays LCD conectados al harware Wiring.

print()
Escribe un int, float, byte, char, char[] o un número con base decimal, hexadecimal, octal o binaria en el display.

println()
Funciona como el método print pero imprime un caracter de nueva linea ('\n') cada vez que es llamado.

clear()
Limpia el display.

home()
Asigna el cursor al principio de la pantalla.

setCursor()
Mueve el cursor para a un posición en linea, columna específica.
