Librerías

MenuItem

 

MenuItem
Item básico con un nombre y un mnemónico opcional.

registerBackend()
Registra un "backend" para este item para usar para retornos (callbacks) y similares.

getValue()
Obtiene el valor de este item.

getShortkey()
Obtiene la abreviación de este item.

increment()
Incrementa el valor de este item.

decrement()
Disminuye el valor de este item.

setLevel()
Asigna el nivel del item.

getLevel()
Obtiene el nivel del item.

hasShortKey()
Revisa si este item tiene una abreviación.

hasChildren()
Revisa si este item tiene hijos (children).

getShortkey()
Obtiene la abreviación del item.

getName()
Obtiene el nombre del item.

getBack()
Obtiene el item 'anterior' de este item (anterior históricamente).

getBefore()
Obtiene el item 'anterior' a este item.

getRight()
Obtiene el item a la derecha de este item.

getAfter()
Obtiene el item siguiente de este item.

getLeft()
Obtiene el item a la izquierda de este item.

add()
Adiciona un item después de este item en jerarquía.

addBefore()
Adiciona el item anterior de este item en jerarquía.

addRight()
Adiciona el item a la derecha de este item en jerarquía.

addAfter()
Adiciona el item después de este item en jerarquía.

addLeft()
Adiciona el item a las izquierda de este item en jerarquía.

isEqual()
Revisa si el item es igual a este.

setBack()
Asigna el item que pertenese como el anterior 'históricamente' para este item.
