SoftwareSerial

La librería de SoftwareSerial de Wiring permite facilmente leer y escribir datos en y desde una maquina externa a través del puerto serial por software. Permite comunicar dos maquinas. Nota: La clase SoftwareSerial no tiene el método available().

El puerto serial es un puerto de nueve pines I/O que existen en la mayoria de los PCs y puede ser emulado a través del USB en Macintosh y PCs con el adaptador USB Serial

 

SoftwareSerial
Clase SoftwareSerial de Wiring (tipo de dato).

begin()
Abre el puerto serial para leer y escribir.

print()
Escribe datos (int, float, byte, char, char[], número base (DEC, BIN, OCT or HEX) o Strings al puerto serial.

println()
Funciona como el método print pero imprime un caracter new line al final cada vez que se llama.

read()
Retorna un número entre 0 y 255 para el siguiente byte esperado en el buffer. Retorna -1 si no hay byte.

flush()
Vacia el buffer del puerto serial. Posteriormente llamando read() retornará datos recibidos después que comando flush() es llamado.
