Librerías

FSM (Finite State Machines)

 

FSM
Asigna el estado inicial de esta Máquina de Estado Finito (Finite State Machine).

transitionTo()
Posee la planificación de las transiciones del FSM para el siguiente update().

update()
Actualiza la FSM.

immediateTransitionTo()
Posee la transición de la FSM al siguiente estado.

getCurrentState()
Obtiene el estado actual de esta FSM.

isInState()
Revisa si esta FSM está en el estado State.

timeInCurrentState()
Obtiene el estado actual de esta FSM.

State
Asigna el estado incial de State.

enter()
Vuelve a llamar la función userEnter.

update()
Vuelve a llamar la función userUpdate.

exit()
Vuelve a llamar la función userExit.
