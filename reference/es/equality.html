Nombre

== (equality)
Ejemplos 
int a = 23;
int b = 23;
if (a == b) {
  Serial.print("las variables a y b son iguales");
}
Descripción Determina si dos valores son equivalentes. El operador de igualdad es diferente del operador de asignación.
Sintaxis 
valor1 == valor2
Parámetros
valor1 int, float, char, byte, boolean
valor2 int, float, char, byte, boolean
Uso Application
Relacionados > (greater than)
< (less than)
>= (greater than or equal to)
<= (less than or equal to)
!= (inequality)
